## 주요 버그 수정 이력
1. CSS 클래스 cl-card→nb-card (에이전트 생성 파일 수정)
2. GAS 데이터 파싱 안전화 (NaN/undefined 방지)
3. Chrome 비밀번호 경고 제거 (type=text로 변경)
4. **방 자동생성 버그** (2회 발생, 근본 수정 완료)
   - 프론트: ROOM CREATE 버튼 방식으로 변경
   - GAS: getRoomSheet() 읽기전용 함수 분리
   - GAS: listRooms() 3가지 접두사 모두 검색
   - GAS: Code.gs 전면 재작성 (KIM'S BACK 패턴 - 헤더행, ensureHeaders 등)
5. 점수 표시 색상 수정 (네이비 배경 흰색 텍스트)
6. 틀린그림 좌표 수정 + 반경 확대 (태블릿 터치 최적화)
